# Unit Testing

Code พื้นฐาน Unit Testing ใช้สำหรับสอนทีม

## เริ่มต้นทดสอบ

หลังจาก install แล้ว เราสามารถ Run Test โดยใช้คำสั่ง

```bash
npm run test
```
